# Actualizar Datos del RUC para Persona Jurídica - Orientación - Superintendencia Nacional de Aduanas y de Administración Tributaria - Plataforma del Estado Peruano

Actualizar Datos del RUC para Persona Jurídica
Descripción
Es el proceso para actualizar o modificar los datos de tu RUC.
El Registro Único de Contribuyentes (RUC) es el número que identifica como contribuyente a una Persona Jurídica o Persona Natural y contiene información sobre el contribuyente, que podrás actualizar o modificar cuando quieras.
Este trámite es gratuito sin importar la modalidad por la que lo realices.
Podrás actualizar los siguientes datos del RUC
Domicilio Fiscal
Fecha de inicio de actividades
Dar Alta de Registro de Establecimientos Anexos
Dar Baja de Registro de Establecimientos Anexos
Dar Alta, Baja o modificar Representantes Legales
Tus datos de celular y correo electrónico
Suspensión de actividades
